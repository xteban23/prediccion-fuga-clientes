# Modelo de Clasificación para la Predicción de Fuga de Clientes en Marketplaces

![Banner](https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcS3AckCCA8nDiYuPqW0XX_U7EonVm_YfNWVNw&s)

## Autores
Santiago Suarez Magon, Esteban Alcardy Pacheco

## Objetivo
Desarrollar un modelo de aprendizaje supervisado (clasificación) que identifique clientes con baja probabilidad de retorno basándose en su historial de compras y calificación de pedidos anteriores.

## Dataset
**Brazilian E-Commerce Public Dataset by Olist**  
Información con link de descarga: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

El dataset contiene aproximadamente 100.000 pedidos realizados entre 2016 y 2018 en múltiples plataformas de comercio electrónico de Brasil. Tiene un peso total de 126 MB y está compuesto por 9 archivos CSV con información sobre pedidos, clientes y geolocalización.

## Modelos
Clasificación, Aprendizaje Supervisado, Random Forest, Regresión Logística, XGBoost, Feature Engineering, Churn Prediction
